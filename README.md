# tensorflow-DSMM

Ongoing project for implementing various Deep Semantic Matching Models (DSMM). DSMM is widely used for:

- search relevance
- question answering
- duplicate detection
- ...

# Supported Models
- DSSM
- CDSSM
- MatchPyramid
- BCNN
- ABCNN
